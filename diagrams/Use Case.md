# Глоссарий

| Термин | Определение |
|:--|:--|
| Пользователь | Человек, использующий приложение для заказа пиццы |
| Гость | Пользователь, который использует приложение без регистрации |
| Зарегистрированный пользователь | Пользователь с учётной записью в приложении |
| Администратор | Управляет меню и заказами |
| Заказ | Сформированная корзина с выбранными товарами и адресом доставки |
| Корзина | Содержимое выбранных товаров перед оформлением заказа |
| Промокод | Код, который можно применить для получения скидки |

## 1 Актёры

| Актёр | Описание |
|:--|:--|
| Гость | Может просматривать меню, добавлять товары в корзину, регистрироваться или входить в систему |
| Зарегистрированный пользователь | Может просматривать меню, управлять корзиной, оформлять заказы, редактировать профиль и отслеживать заказы |
| Администратор | Может управлять меню, просматривать заказы и обновлять их статус |

<a name="use_case"/>

## 2 Варианты использования

### 2.1 Войти в учётную запись (Login)

**Актёры:** Гость  
**Описание:** Позволяет пользователю войти в приложение с учётной записью.  
**Предусловия:** Пользователь выбрал функцию входа.  
**Основной поток:**
1. Приложение отображает окно входа;
2. Пользователь вводит данные для входа;
3. Приложение проверяет данные и присваивает статус "Зарегистрированный пользователь";
4. Вариант использования завершается.

---

### 2.2 Зарегистрироваться (Register)

**Актёры:** Гость  
**Описание:** Позволяет пользователю создать учётную запись.  
**Предусловия:** Пользователь выбрал функцию регистрации.  
**Основной поток:**
1. Приложение отображает окно регистрации;
2. Пользователь вводит данные и подтверждает;
3. Приложение создаёт новую учётную запись;
4. Вариант использования завершается.

---

### 2.3 Войти как гость (Guest Login)

**Актёры:** Гость  
**Описание:** Позволяет пользователю использовать приложение без регистрации.  
**Основной поток:**
1. Пользователь выбирает опцию "Войти как гость";
2. Приложение присваивает статус "Гость";
3. Вариант использования завершается.

---

### 2.4 Просмотреть меню (View Menu)

**Актёры:** Гость, Зарегистрированный пользователь  
**Описание:** Позволяет пользователю просмотреть список доступных пицц и товаров.  
**Основной поток:**
1. Пользователь выбирает просмотр меню;
2. Приложение отображает все товары;
3. Вариант использования завершается.

---

### 2.5 Добавить товар в корзину (Add to Cart)

**Актёры:** Гость, Зарегистрированный пользователь  
**Описание:** Позволяет добавить выбранный товар в корзину.  
**Предусловия:** Пользователь просматривает меню.  
**Основной поток:**
1. Пользователь выбирает товар;
2. Пользователь добавляет его в корзину;
3. Вариант использования завершается.

---

### 2.6 Редактировать корзину (Edit Cart)

**Актёры:** Зарегистрированный пользователь  
**Описание:** Позволяет изменять содержимое корзины.  
**Предусловия:** Корзина содержит товары.  
**Основной поток:**
1. Пользователь открывает корзину;
2. Пользователь изменяет количество товаров или удаляет их;
3. Вариант использования завершается.

---

### 2.7 Оформить заказ (Place Order)

**Актёры:** Зарегистрированный пользователь  
**Описание:** Позволяет оформить заказ на выбранные товары.  
**Предусловия:** Корзина содержит товары.  
**Основной поток:**
1. Пользователь подтверждает заказ;
2. Приложение формирует заказ и сохраняет данные доставки;
3. Вариант использования завершается.

---

### 2.8 Отследить заказ (Track Order)

**Актёры:** Зарегистрированный пользователь  
**Описание:** Позволяет отслеживать статус заказа.  
**Предусловия:** Пользователь оформил заказ.  
**Основной поток:**
1. Пользователь выбирает заказ для отслеживания;
2. Приложение отображает текущий статус;
3. Вариант использования завершается.

---

### 2.9 Редактировать профиль (Edit Profile)

**Актёры:** Зарегистрированный пользователь  
**Описание:** Позволяет редактировать данные профиля.  
**Предусловия:** Пользователь вошёл в систему.  
**Основной поток:**
1. Пользователь открывает окно редактирования профиля;
2. Пользователь изменяет данные и сохраняет;
3. Вариант использования завершается.

---

### 2.10 Выйти из учётной записи (Logout)

**Актёры:** Зарегистрированный пользователь  
**Описание:** Позволяет выйти из приложения.  
**Основной поток:**
1. Пользователь выбирает функцию выхода;
2. Приложение завершает сеанс пользователя;
3. Вариант использования завершается.

---

### 2.11 Управлять меню (Manage Menu)

**Актёры:** Администратор  
**Описание:** Позволяет добавлять, изменять и удалять товары меню.  
**Предусловия:** Администратор вошёл в систему.  
**Основной поток:**
1. Администратор открывает меню управления;
2. Вносит изменения;
3. Вариант использования завершается.

---

### 2.12 Просматривать заказы (View Orders)

**Актёры:** Администратор  
**Описание:** Позволяет просматривать список всех заказов.  
**Предусловия:** Заказы существуют в системе.  
**Основной поток:**
1. Администратор выбирает просмотр заказов;
2. Приложение отображает список всех заказов;
3. Вариант использования завершается.

---

### 2.13 Обновлять статус заказа (Update Order Status)

**Актёры:** Администратор  
**Описание:** Позволяет изменять статус заказов.  
**Предусловия:** Заказы существуют в системе.  
**Основной поток:**
1. Администратор выбирает заказ;
2. Изменяет его статус;
3. Вариант использования завершается.
